介绍 女性编程日

HTML & CSS 工作坊介绍

About this workshop

Dive head first into HTML & CSS. This workshop takes a project-based approach to HTML & CSS. By distinguishing good code from bad, participants will focus on learning a subset of the good parts in order to make something (quick and) awesome.

Tackle the barebones of how to quickly develop a web site, and get it live on the Web by the end of the workshop.

Prereps & Preparation

Basic HTML experience may be helpful, but is not necessary. Bring a laptop to class (Mac preferred but not required). Please download and install Chrome and Atom (Hint: the download button is on the home page). Please email us prior to class if you have any additional concerns or questions.

    F.A.Q.

    常见问题

    如何选择参与者

    因为名额有限，我们仅面向在校大学生和职场女性（全职妈妈也是在职女性哦）。请详细填写申请表单，向我们表达「为何你会是一个好的参与者」。我们希望选择有能力，也有意愿帮助她人的候选者。

    编程小白也可以参加吗

    本次Workshop不需要你有任何的编程基础亦可参加，工作坊的目的就是启发你的编程基因，希望你找到更多「可能的自我」

    男性可以参加此次工作坊吗

    我们关注如何帮助更多女性如何进入到IT世界，在男女比例验证失衡的IT领域我们希望通过这些活动增加女性比例。当然，我们坚信任何人都应该学习编程，你可以通过我们的合作伙伴找到一些其他学习资源，例如思特沃克学院。

    我们会学会什么

    我们主要学习如何制作并部署一个静态网站，本网站就是一个静态网站，用来展示你的Idea。

    I’d like to help out!

    当然！本次活动是基于志愿者行为，你可以帮助我们宣传活动，过来当志愿者，还有甚至在当地组织workshop。
